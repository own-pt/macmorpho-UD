Brief report about results on training OpenNLP's models on MacMorpho and Bosque data.

Available models are:

Maxent
Maximum Entropy (Generalized Iterative Scaling method)
MaxentQN
Maximum Entropy (with QuasiNewton method)
NaiveBayes
Naive Bayes
Perceptron
Perceptron (binary, it seems)
PerceptronSequence
Perceptron for sequences (unclear from documentation) Although is listed, it isn't working: it gives a NullPointerException

We ran 2 experiments:

Experiment 1
For each OpenNLP model and each Macmorpho tagset version (UD, UD+PCP, MM), we trained in the train+dev divisions, and evaluated in test division. Our hypothesis is that coarser datasets are easy to learn.
Experiment 2
We trained the tagger with macmorpho UD (without PCP), using it entirely as training set (train+dev+test). We evaluated it in Bosque (test). We compared our result to the tagger trained in Bosque (train+dev) and evaluated in Bosque (test).

2 Execution

Trained in BRL cluster.

Scripts and data are in the repository (branch experiment): /macmorpho/opennlp/training-files

Raw result files as well: /macmorpho/opennlp/results

For training for both experiments, the following steps were taken:

  1. Merging train and dev (OpenNLP does not use held-out (validation) data in training, it seems) 

    for i in {keep-pcp,mm-revisto,remove-pcp} ; do cat macmorpho-v1-dev-$i.conllu >> macmorpho-v1-train-$i.conllu ; mv macmorpho-v1-train-$i.conllu macmorpho-v1-train-and-dev-$i.conllu ; done

  2. Producing files with complete data (train + dev + test) in order to compare with Bosque (experiment 2) 

    for i in {keep-pcp,mm-revisto,remove-pcp} ; do
    cat macmorpho-v1-train-and-dev-$i.conllu >> macmorpho-v1-full-$i.conllu ;
    cat macmorpho-v1-test-$i.conllu >> macmorpho-v1-full-$i.conllu ;
done
  3. Scripts suffixed by -complete-data created in order to train with full data as well.

  4. Train with:

    • run-params.sh
    • run-params-complete-data.sh

    I'm using these in order to use train different models.

A similar procedure was done for Bosque (UDPortuguese, official 2.0 release), in order to train it with OpenNLP:

  1. Merging train and dev
  2. Scripts suffixed by -bosque in order to train with bosque.
  3. Train with run-params-bosque.sh.

For each dataset, one model is trained for each *TrainerParams.txt file: Maxent, MaxentQN, NaiveBayes, Perceptron, PerceptronSequence
